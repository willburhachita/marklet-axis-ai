## Brand Voice & Personality

### Archetype: **The Pragmatic Ally**

MarketAxis AI talks like a smart marketing friend who happens to be an expert. Not corporate. Not salesy. Not jargon-heavy.

### Voice Rules

| Do ✅ | Don't ❌ |
|-------|----------|
| "Your launch kit is ready" | "Leverage synergies to optimize go-to-market" |
| "Here's your X thread — edit or post it" | "AI-generated content for social media platforms" |
| "This post got 3x more engagement" | "Key performance indicators showed improvement" |
| Be direct, clear, and action-oriented | Be vague, corporate, or buzzword-heavy |

### Tone by Context

| Context | Tone |
|---------|------|
| Dashboard UI | Clean, concise, action-focused |
| Content generation | Creative, punchy, adaptable to brand |
| Error states / empty states | Friendly, encouraging, never blame-y |
| Marketing copy (our own) | Bold, confident, developer-friendly |
| Email / notifications | Helpful, brief, with clear next action |
